Password Generator

This project is developed as part of the Oasis Infobyte Python Programming Internship.

Description:
The Password Generator is a Python program that generates a random and secure password based on user-defined preferences.

Features:
- User-defined password length
- Option to include lowercase letters
- Option to include uppercase letters
- Option to include numbers
- Option to include special characters

Technology Used:
Python

Working:
The program takes input from the user regarding password length and character set preferences, then generates a random password using the selected character sets.
