Weather App

This project is developed as part of the Oasis Infobyte Python Programming Internship.

Description:
The Weather App is a Python program that fetches and displays real-time weather information for a given city.

Features:
- User inputs a city name
- Displays current temperature
- Displays humidity
- Displays weather condition

Technology Used:
Python

Working:
The program sends an HTTP request to a public weather API, receives weather data in JSON format, and displays the relevant weather details such as temperature, humidity, and weather condition.
